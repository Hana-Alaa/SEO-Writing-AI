STRICT WRITING RULES
1. Language lock:
   - Write strictly in {{ article_language }}.
   1.1 Perspective & Pronouns (MANDATORY):
      - Address the reader directly in SECOND PERSON in the target language.
      - Use natural second-person pronouns:
      Arabic: "أنت / لك / لديك / عملك / شركتك"

      French: "vous / votre"
      German: "Sie / Ihr"

      - At least one second-person construction in each paragraph.

2. Structure:
   - Base structure: 2-3 short paragraphs per section.
   - PARAGRAPH LENGTH LIMIT (CRITICAL): Each paragraph MUST be exactly 3 to 4 lines maximum (roughly 2-3 sentences). 
   - DO NOT write long "wall-of-text" paragraphs. If a paragraph exceeds 4 lines of thought, break it into a new paragraph immediately.
   - If requires_list = true: add ONE markdown list (3–5 items) after paragraph 2.
   - If requires_table = true: add ONE markdown table (3+ rows) after paragraph 2.
   - CRITICAL TABLE RULE: Every table row MUST end with a `|` pipe character and NOTHING ELSE.
     - CORRECT: `| Cell 1 | Cell 2 | Cell 3 |`
     - WRONG:   `| Cell 1 | Cell 2 | Cell 3 | Some extra text here`
     - After the last `|` of any table row, the line MUST end immediately with no additional text.
     - A table MUST start on a NEW LINE and MUST be preceded by a blank line.
     - Any continuation text MUST be on a new separate paragraph AFTER a blank line.
   - FAQ sections keep H3 Q&A format only.

3. Keyword usage (Density target: 1.2%–1.6% of section word count):
   - For a ~150-word section: use primary keyword 2–3 times naturally.
   - First mention MUST be in the first paragraph.
   - At least one mention should be bolded (**keyword**) — but ONCE only per section.

5. CTA policy:
   - CTA sentence MUST directly address the reader in second person.
   - Decide based on CTA Type assigned to this section:
     * "none"     → Write NO CTA at all.
     * "soft"     → One low-pressure sentence (e.g. "Explore our full guide", "Learn more about our approach"). No hard sell.
     * "primary"  → One clear action CTA in the FIRST PARAGRAPH (e.g. "Request a free consultation today", "Start your project with us"). Wrap in brand link if Brand Link Used = false.
     * "moderate" → One action CTA at the END of the section. Can be: service mention, offer, contact/WhatsApp invite, product link. Wrap in a relevant URL if available.
     * "strong"   → One high-urgency, conversion-focused CTA as the VERY LAST SENTENCE only (e.g. "Request a free consultation today", "Get your tailored proposal", "Book your strategy session", "Call us now"). Do NOT use purely motivational closings. Wrap in brand link if Brand Link Used = false.
   - CTA tone depends on intent:
     * commercial/transactional → direct, benefit-driven, action-urgent, conversion-focused.
     * informational → educational, resource-oriented, no pressure.
   - STRICT ANCHOR TEXT RULE: Never use weak, generic anchor texts like "Let's talk", "Contact Us", "See All Posts", or "Click Here". 
     Instead, the linked text MUST be highly descriptive, benefit-driven, and context-specific (e.g., "Get Your Custom Web Design Proposal", "Schedule Your Free SEO Strategy Session").
   - If Brand Link Used = false: wrap the CTA text as a markdown link using: {{ brand_url }}
   - If Brand Link Used = true: write the CTA without adding the brand link again.
   - Never output "(None)" or a broken link. If you have no valid URL, write the CTA as plain text.
